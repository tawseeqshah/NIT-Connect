# NIT Connect Proposal App

NIT Connect is a Flutter-based app designed to facilitate communication and collaboration within an institution. It serves as a platform for users to submit their ideas, suggestions, queries, and reports across various categories. Whether it's proposing improvements for sports facilities, sharing feedback about the hostel and mess services, addressing e-governance concerns, suggesting library enhancements, or raising issues related to accounts and alumni, NIT Connect provides a unified space for users to engage with the relevant sections and departments.

## Features

- Multiple categories: NIT Connect supports a diverse range of categories, ensuring that users can address their specific concerns and requirements.
- Seamless communication: The app enables users to communicate their ideas and suggestions, fostering collaboration between the user community and the respective departments.
- User-friendly interface: NIT Connect offers an intuitive and user-friendly interface that enhances the user experience and makes it easy for users to navigate through different sections.
- Dark mode: The app provides a dark mode option, allowing users to choose their preferred theme and personalize their experience.
- Streamlined sections: Each category within the app has its own dedicated section screen, tailoring the available actions and options to the specific needs of that category.
- Proposal submission: Users can create proposals by sharing their innovative ideas and suggestions related to a particular category, contributing to the ongoing improvement and development of the institution.
- Query submission: Users can ask questions and seek answers related to a specific category, promoting knowledge sharing and facilitating problem-solving.
- Section rating: NIT Connect enables users to provide feedback and ratings for each category, helping identify areas of improvement and recognize departments' efforts.
- Issue reporting: Users can report any issues or violations they come across within a category, ensuring a proactive approach to resolving problems.


## Getting Started

To run the app on your local machine, follow these steps:

1. Ensure that you have Flutter and Dart installed on your system.
2. Clone this repository to your local machine.
3. Open the project in your preferred Integrated Development Environment (IDE) or text editor.
4. Run the command `flutter pub get` to fetch the required dependencies.
5. Connect your device or launch an emulator.
6. Run the command `flutter run` to start the app.

## Dependencies

This project utilizes the following dependencies:

- Flutter SDK
- material.dart
- cupertino_icons.dart

Before running the app, ensure that these dependencies are correctly included in your `pubspec.yaml` file.
